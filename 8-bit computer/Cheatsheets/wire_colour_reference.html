Wire Colour Reference

// Ben Eater 8-bit Computer Build — Marcus

Red +5V Power

Any wire carrying +5V supply. Never reuse for signals.

  • Power rail to pin 8 (Vcc) on each 555 timer
  • Power rail to pin 14 (Vcc) on 74LS logic chips
  • Power rail to pin 4 (Reset) on 555 — tied permanently high
  • PN Labs Protect Nano 5V output to main power rail
Black GND

Any wire carrying ground. Never reuse for signals.

  • Pin 1 (GND) on each 555 timer to ground rail
  • Pin 7 (GND) on each 74LS logic chip to ground rail
  • Negative leg of clock LED to ground via 220Ω resistor
  • Decoupling capacitor negative legs to ground rail
Yellow 8-bit Data Bus

All 8 bus lines, consistently yellow across every module.

  • Not present in clock module — clock produces timing, not data
  • A register output lines to central bus (8 wires, one per bit)
  • B register output lines to central bus
  • ALU result lines to central bus
Blue Clock Signal (CLK)

The CLK line wherever it travels across the board.

  • CLK output from U5C / U4F leaving the clock module
  • CLK wire running to registers, program counter, RAM
  • CLK̅ (inverted clock) line where needed
  • Astable 555 output into logic gate network
Green Control Signals

Wires giving a module permission to act. Ask: "is this enabling something?"

  • HLT input to clock module — halts clock from CPU
  • Bistable output switching auto / manual clock mode
  • Write enable (WE) into RAM module
  • Output enable (OE) lines on register chips
White Module-Internal Wiring

Stays within one module's area. A long white wire is a warning sign.

  • Pin 2 and pin 6 of astable 555 joined together
  • Junction between 1kΩ fixed resistor and potentiometer
  • Astable 555 output into AND gate (U5A) pin 2
  • SR flip-flop connections inside the bistable circuit
// Golden Rules Red and Black are sacred — never reuse for signals under any circumstances.
Yellow bus lines must be consistent across every module on the board.
If short of a colour mid-module, borrow White — never raid power or bus colours.
Cirkit uses light blue for GND — ignore it. Your physical standard is what counts.
